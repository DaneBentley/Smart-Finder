2. Description of Service

Smart Finder is a Chrome extension that provides AI-powered search capabilities to help users find specific content on web pages using natural language queries or traditional text search.

2.1 Core Features

  • Traditional Search: Free, unlimited text-based search functionality with multi-term highlighting
  • Regex Search: Free, unlimited advanced pattern matching with automatic pattern detection
  • Pattern Detection: Automatic recognition of emails, phone numbers, dates, URLs, and other common patterns
  • AI-Powered Search: Natural language queries processed using Groq Llama 3.1 8B Instant
  • Multi-Color Highlighting: Up to 5 different colors for distinguishing multiple search terms
  • Shadow DOM UI: Isolated user interface that doesn't interfere with website styling
  • Dynamic Content Monitoring: Automatic detection and re-search of dynamically loaded content
  • Token System: Fair usage credit-based system for AI search functionality
  • Google Authentication: Secure sign-in via Google OAuth 2.0
  • Payment Integration: Stripe-powered secure token purchasing system
  • Custom AI Prompts: User-configurable AI behavior and instructions

2.2 Token System

  • AI searches require tokens (search credits)
  • Free tokens are provided monthly with automatic reset cycles
  • Additional tokens may be purchased through our Stripe payment system (100 tokens = $1.00)
  • Token consumption varies based on content length and search complexity - longer pages may require multiple tokens per search
  • Tokens are only consumed when AI search finds and returns relevant matches
  • Failed AI searches or searches with no results do not consume tokens
  • Traditional text search and regex search are always free and unlimited
  • Unused tokens do not expire but free tokens reset monthly
