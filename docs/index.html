🚀 Powerful Features

🤖
AI-Powered Search
Ask questions in natural language like "find contact information" or "show me prices" and get intelligent results powered by Groq Llama 3.1.
🎯
Smart Pattern Detection
Automatically detects emails, phone numbers, URLs, dates, addresses, and more. Just type "email" to find all email addresses instantly.
🌈
Multi-Color Highlighting
Highlight up to 5 different search terms simultaneously with distinct colors. Perfect for comparing information or tracking multiple data points.
⚡
Lightning Fast
Instant search results with real-time highlighting. Works on dynamic content and single-page applications seamlessly.
🔒
Privacy Focused
Traditional searches are processed locally. AI searches are processed securely with no permanent data storage.
🎨
Non-Intrusive UI
Shadow DOM isolation ensures the search interface never conflicts with website styling or functionality.
